# 🌸 Zero Two Web

Website bertema **Zero Two** – karakter ikonik dari anime *Darling in the Franxx*.
Proyek ini dibuat sebagai situs sederhana namun stylish yang menampilkan informasi, galeri, dan nuansa estetika cyber–anime.

---

## 🎯 Tujuan Proyek

* Menyediakan website fanpage bertema Zero Two
* Menampilkan desain modern, responsif, dan menarik
* Mengimplementasikan next.js secara kreatif
* Memberikan pengalaman visual bertema anime

---

## ✨ Fitur Utama

* **Responsive Design** – tampilan menyesuaikan desktop & mobile
* **Interactive UI** – animasi dan efek modern
* **Gallery Section** – koleksi gambar Zero Two
* **Character Info** – deskripsi dan fakta menarik
* **Smooth Navigation** – navigasi cepat dan ringan

---

## 🌟 Preview

Website ini dirancang dengan tema warna khas Zero Two:

* Pink
* Merah muda
* Hitam
* Aksen futuristik

Tampilan minimalis namun tetap berkarakter kuat.
